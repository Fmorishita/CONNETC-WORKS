# Service card photos ("What We Do" section)

Drop a **JPG** with **exactly** one of these names in this folder to show a real
photo on that service card. If a file isn't here yet, the card automatically
falls back to its icon design — so nothing ever looks broken.

| File name (use exactly) | Service card |
|---|---|
| `video-surveillance.jpg` | Video Surveillance |
| `access-control.jpg` | Access Control |
| `intercom.jpg` | Intercom & Communication |
| `structured-cabling.jpg` | Structured Cabling |
| `fiber-optics.jpg` | Fiber Optics |
| `network-wireless.jpg` | Network & Wireless |
| `commercial-av.jpg` | Commercial Audio & Video |

**Tip:** landscape images ~800×500px, under ~300 KB each, look best.
Upload here on GitHub with **Add file ▸ Upload files**, then commit.
