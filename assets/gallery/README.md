# Project gallery photos

Drop a **JPG** with **exactly** one of these names in this folder (`assets/gallery/`)
to replace a gallery placeholder. Until you upload one, a placeholder graphic
shows automatically.

> These names use a **`project-`** prefix so they're never confused with the
> Services photos (which live in `assets/services/`).

| File name (use exactly) | Gallery tile |
|---|---|
| `project-cabling.jpg` | Structured Cabling |
| `project-cameras.jpg` | Security Cameras |
| `project-access.jpg` | Access Control |
| `project-intercom.jpg` | Intercom |
| `project-network.jpg` | Network & Wireless |
| `project-team.jpg` | Service Vehicle / Local Team |

**Tip:** landscape ~1200×900px, keep each under ~400 KB (compress before upload).
Upload here on GitHub with **Add file ▸ Upload files**, then commit.
